Alloy Strength Training Systems

https://www.alloystrengthtraining.com

A subscription-based membership site built for 2 personal trainers to implement their unique training system based on progressive exercises.

Alloy Strength Training has a user and admin-side that allows the trainers to upload videos and determine exercises that new members user can progress through in their workouts. The program has an algorithm built in that calculates a "suggested weight" based on each new workout result that a user submits.
